# Vk_toxic_comments_classification
Описание:
Программа для классификации опасных комментариев в сети ВКонтакте. Она автоматически определяет спам, оскорбления, экстремизм и другие нарушения в комментариях. Проект сделан в учебных целях.

Использовалось:
Python, VK API, sqlite3, Ollama (Llama 3), nltk.stem.snowball.RussianStemmer, Navec, matplotlib, seaborn, а также другие Python-библиотеки для обработки и анализа данных.

Функционал / Как работает:
Сначала программа собирает комментарии из постов выбранного сообщества ВКонтакте. После этого комментарии фильтруются, и на выходе формируется датасет примерно из 3000 комментариев. Далее создаётся CSV-файл, где сохраняются сами комментарии, ID пользователей и другая связанная информация.

Затем с помощью автоматически сформированного промпта и LLM-модели каждый комментарий относится к одной из категорий. После этого создаётся отдельный CSV-файл уже с результатами классификации.

Также в проекте есть два дополнительных способа анализа. Первый — через поиск маркерных слов, которые характерны для определённых категорий комментариев. По результатам этого анализа создаётся один CSV-файл. Второй способ — анализ через эмбеддинги слов с использованием модели Navec, по которому тоже формируется отдельный CSV-файл.

В конце результаты разных подходов сравниваются между собой.

Для запуска:
Нужно создать Python-проект, установить необходимые библиотеки и открыть Jupyter Notebook. Для работы программы также необходимо указать свой токен ВКонтакте.
